Panduan Belajar HTML untuk Pemula

Tentang Buku

Buku HTML Uncover ini ditujukan untuk pemula yang baru mulai belajar web programming. Tidak ada skill khusus yang disyaratkan, saya akan membahas semuanya dari dasar, mulai dari cara menginstall teks editor, cara menjalankan file HTML, hingga pembahasan lanjutan seperti form validation HTML5, canvas, dan SVG. Namun saya berasumsi anda sudah cukup familiar menggunakan web browser seperti Google Chrome atau Mozilla Firefox. Jika belum pernah mempelajari kode HTML sebelumnya, saya sarankan untuk mengikuti bab dalam buku ini secara berurutan, mulai dari bab 1 hingga akhir. Pembahasan dalam sebuah bab bisa saja menggunakan materi dari bab sebelumnya
